# Python Student Grade Analytics

This is a simple academic Python project for analyzing student grades and performance.

## About

The project reads student marks and shows basic analysis such as average marks, highest marks, lowest marks, grade category, and performance summary.

## Features

* Calculate average marks
* Find highest and lowest marks
* Show grade summary
* Analyze student performance
* Simple and beginner-friendly code

## Technologies Used

* Python
* CSV file handling
* Basic data analysis

## What I Learned

Through this project, I practiced Python programming, data handling, basic statistics, and organizing an academic project .
